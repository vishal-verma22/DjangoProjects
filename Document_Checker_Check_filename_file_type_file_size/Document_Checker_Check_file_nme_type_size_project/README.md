**# Document Format Checker**



**A Django web application to validate uploaded documents for correct format, file type, size, and naming conventions. It allows users to upload files such as mark sheets and ID cards, checks for errors, and provides success/error feedback.**



**## Features**



**- Upload multiple documents (Marksheet, ID Card) via a user-friendly form**

**- Validates:**

  **- File type (`jpg`, `pdf`, `docx`, `txt`)**

  **- File size (max 8MB)**

  **- No spaces in file names**

**- Displays detailed error messages or success confirmation**

**- Built with Django forms and templates**

**- Styled with CSS for a clean and responsive UI**



**## Tech Stack**



**- \*\*Backend:\*\* Python, Django**  

**- \*\*Frontend:\*\* HTML, CSS**  

**- \*\*Other Libraries:\*\* None (uses Django built-in features)**
