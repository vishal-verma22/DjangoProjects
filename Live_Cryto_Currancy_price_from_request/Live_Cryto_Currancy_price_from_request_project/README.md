**# Live Crypto Currency Price App**



**A Django-based web application that fetches and displays live cryptocurrency prices**

**in INR using the CoinGecko public REST API. This project demonstrates real-world**

**backend development concepts such as API integration, JSON handling, and dynamic**

**template rendering.**



**---**



**## Features**

**- Fetches real-time cryptocurrency market data**

**- Displays market rank, coin name, image, and current price**

**- Server-side rendering using Django templates**

**- Secure POST request with CSRF protection**

**- Clean and simple UI using HTML and CSS**



**---**



**## Tech Stack**

**- Backend: Python, Django**

**- Frontend: HTML, CSS**

**- API: CoinGecko REST API**

**- Library: requests**
