**# Student Management System Django Project**



**## Overview**

**This is a simple \*\*Student Management System\*\* built with \*\*Django\*\*. It allows users to \*\*create, view, update, and delete student records\*\*. The project demonstrates the use of Django \*\*Models, Forms, and Views\*\*, including validation and basic CRUD operations.**

**## Features**

**- \*\*Create Student\*\* – Add a new student with name, roll number, subject, and marks.**

**- \*\*View Students\*\* – Display a list of all students in the system.**

**- \*\*Update Student\*\* – Edit student details. Roll number is read-only during updates.**

**- \*\*Delete Student\*\* – Remove a student record from the database.**

**- \*\*Form Validations\*\* –** 

  **- Name must have at least 2 characters.**

  **- Roll number must be a positive integer.**



**---**



**## Tech Stack**

**- \*\*Backend:\*\* Django (Python)**

**- \*\*Database:\*\* SQLite (default Django DB)**

**- \*\*Forms:\*\* Django ModelForms with custom validation**

**- \*\*Templates:\*\* HTML (basic CRUD pages)**

**- \*\*Frontend:\*\* Minimal CSS for form styling**
